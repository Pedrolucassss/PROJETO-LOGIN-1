# PROJETO-LOGIN-1

## Sobre o projeto
Este projeto consiste na criação de uma tela de login utilizando **HTML**, **CSS** e **Bootstrap**.  
O objetivo é praticar conceitos de front-end, responsividade e documentação de projetos.

## Tecnologias utilizadas
- HTML5
- CSS3
- Bootstrap

## Funcionalidades
- Campos de login e senha
- Botão de acesso
- Mensagens traduzidas para português
